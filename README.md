webpack4 多入口，多页面项目构建案例。
=====================================

#webpack 版本需大于4.1
##热更新 ES6转义 代码压缩 SASS AXIOS项目开发必需品均有 可直接进行从0自定义开发


```npm
npm run c page(页面名称) 
```
#动态添加入口，自动化配置页面


```npm
npm/cnpm install //依赖
npm run dev //开发环境
npm run build //生产环境打包
```
